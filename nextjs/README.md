# Next.js Serverless Backend & Frontend

โฟลเดอร์สำหรับพัฒนาเว็บแอปพลิเคชันและ API ด้วย Next.js ร่วมกับ Drizzle ORM

## โครงสร้างและการเชื่อมต่อ
- **Serverless API:** ทำหน้าที่รับข้อมูลอุณหภูมิและความชื้นจาก ESP32-C3
- **Database:** เก็บข้อมูลสภาพอากาศ
- **Drizzle ORM:** จัดการ Schema และ Query ข้อมูลสภาพอากาศมาแสดงที่ Frontend

*(โปรเจกต์นี้จะเริ่มพัฒนาในเฟสถัดไปหลังจากทดสอบฮาร์ดแวร์ฝั่ง ESP32-C3 สำเร็จ)*
